## Введение

Калькулятор предназначен для врачей-генетиков, диетологов, педиатров и неонатологов, работающих с пациентами с нарушениями митохондриального β-окисления жирных кислот (FAOD).

### Поддерживаемые заболевания (МКБ E71.3)

| Код | Название | Ген |
|-----|----------|-----|
| OCTN2 | Первичный дефицит карнитина | SLC22A5 |
| CPT1 | Дефицит карнитинпальмитоилтрансферазы 1 | CPT1A |
| CACT | Дефицит карнитин-ацилкарнитин транслоказы | SLC25A20 |
| CPT2 | Дефицит карнитинпальмитоилтрансферазы 2 | CPT2 |
| SCAD | Дефицит короткоцепочечной ацил-КоА дегидрогеназы | ACADS |
| MCAD | Дефицит среднецепочечной ацил-КоА дегидрогеназы | ACADM |
| VLCAD | Дефицит очень длинноцепочечной ацил-КоА дегидрогеназы | ACADVL |
| SCHAD | Дефицит короткоцепочечной 3-ОН ацил-КоА дегидрогеназы | HADH |
| LCHAD | Дефицит длинноцепочечной 3-ОН ацил-КоА дегидрогеназы | HADHA |
| TFP | Дефицит митохондриального трифункционального белка | HADHA/HADHB |
| GA2 | Глутаровая ацидурия тип 2 (MADD) | ETFA/ETFB/ETFDH |

### Основа расчётов

Клинические рекомендации Российской Федерации 2024 года «Нарушения митохондриального β-окисления жирных кислот».

## Запуск калькулятора

### Требования

- Современный веб-браузер (Chrome, Firefox, Safari, Edge)
- Локальный веб-сервер (для загрузки JSON-файлов)

### Способы запуска

**Способ 1: Через локальный сервер Python**

```bash
cd /путь/к/калькулятору
python3 -m http.server 8000
```

Откройте в браузере: `http://localhost:8000`

**Способ 2: Через VS Code Live Server**

1. Установите расширение "Live Server" в VS Code
2. Откройте папку проекта
3. Нажмите "Go Live" в правом нижнем углу

**Способ 3: Через Node.js**

```bash
npx serve .
```

## Ввод данных пациента

### Блок 1: Данные пациента

| Поле | Описание | Обязательно |
|------|----------|-------------|
| Дата рождения | Дата рождения пациента | Да |
| Дата расчёта | Дата, на которую производится расчёт (по умолчанию — сегодня) | Нет |
| Масса тела | Текущая масса в кг (с точностью до 0.1 кг) | Да |
| Пол | Мужской / Женский | Да |

> **Возраст** рассчитывается автоматически и отображается под полями.

### Блок 2: Диагноз

1. **Выберите диагноз** из выпадающего списка
2. Если диагноз имеет несколько клинических форм (CPT2, VLCAD, GA2), появится дополнительное поле для выбора формы

#### Клинические формы

**CPT2:**
- Неонатальная — тяжёлая форма с началом в первые недели
- Инфантильная — манифестация от 6 мес до 2 лет
- Поздняя мышечная — миалгия, рабдомиолиз у взрослых

**VLCAD:**
- Бессимптомная — выявлена при скрининге
- Симптоматическая — с кардиомиопатией, гипогликемией

**GA2:**
- Тяжёлая неонатальная с аномалиями
- Тяжёлая неонатальная без аномалий
- Мягкая/поздняя форма
- Рибофлавин-чувствительная форма

### Блок 3: Текущее состояние

| Состояние | Описание | Коррекция |
|-----------|----------|-----------|
| Стабильное | Обычный режим питания | Базовые расчёты |
| Интеркуррентное заболевание | ОРВИ, гастроэнтерит, лихорадка | Калорийность +10%, интервалы сокращены |
| Метаболический криз | Экстренная ситуация | Калорийность +15%, LCT=0%, интервал 2 ч |

> ⚠️ При выборе «Метаболический криз» появится предупреждение о необходимости госпитализации.

### Блок 4: Дополнительно

Опциональные флаги для уточнения рекомендаций:
- Грудное вскармливание
- Гипогликемия в анамнезе
- Кардиомиопатия
- Поражение печени

## Интерпретация результатов

### Предупреждения

В верхней части результатов отображаются важные предупреждения:
- 🔴 **Опасно** — критичные ограничения (MCT противопоказаны, карнитин противопоказан)
- 🟡 **Внимание** — важные замечания
- 🔵 **Информация** — дополнительные рекомендации

### Энергетическая потребность

| Параметр | Описание |
|----------|----------|
| Суточная калорийность | Общая потребность в ккал/сут |
| На кг массы тела | Расчётная норма ккал/кг |
| Коррекция | Если применена (+10% или +15%) |

### Макронутриенты

Таблица содержит:
- **Белки** — граммы, % от калорийности
- **Углеводы** — граммы, % от калорийности
- **Жиры** (всего) — общее количество
  - **MCT** — среднецепочечные триглицериды
  - **LCT** — длинноцепочечные триглицериды

> ⚠️ Обратите внимание на примечания под таблицей (например, «только незаменимые ЖК» для LCT при LCHAD/TFP).

### Незаменимые жирные кислоты

- Линолевая кислота (LA, омега-6)
- Альфа-линоленовая кислота (ALA, омега-3)
- DHA (только для LCHAD, TFP) — с указанием диапазона доз
- Соотношение LA:ALA (оптимально 5-10:1)

### Режим питания

| Параметр | Описание |
|----------|----------|
| Интервал днём | Максимально допустимый перерыв между кормлениями |
| Интервал ночью | Максимально допустимое ночное голодание |
| Кормлений в сутки | Рекомендуемое количество |

**Рекомендации:** список специфичных указаний для возрастной группы и состояния.

**Кукурузный крахмал:** для детей старше 8 месяцев при склонности к ночной гипогликемии.

### Медикаментозная терапия

| Препарат | Когда назначается |
|----------|-------------------|
| L-карнитин | OCTN2 (обязательно), вторичный дефицит |
| Рибофлавин | GA2 (обязательно) |
| DHA | LCHAD, TFP (обязательно), VLCAD (рекомендуется) |
| CoQ10 | Опционально при GA2, VLCAD, LCHAD, TFP |
| Жирорастворимые витамины | При строгом ограничении жиров |

### Специализированное питание

Для детей до 3 лет отображается:
- Тип вскармливания (100% смесь / смешанное / обычное)
- Объём смеси (мл/кг/сут)
- Рекомендуемые смеси (Нутриген, Моноген и др.)

### Экстренный протокол

При интеркуррентном заболевании или кризе отображается:
- Концентрация раствора глюкозы
- Скорость введения (мл/час)
- Суточный объём
- Инструкции по действиям
- Показания к госпитализации

## Клинические рекомендации по диагнозам

### OCTN2 (Первичный дефицит карнитина)

- **Диета:** Не требуется специальная
- **MCT:** Не противопоказаны
- **Карнитин:** 100-200 мг/кг/сут (основное лечение)
- **Особенности:** Пожизненный приём карнитина

### MCAD

- **Диета:** Не требуется специальная
- **MCT:** ⚠️ ПРОТИВОПОКАЗАНЫ
- **Карнитин:** При документированном дефиците
- **Особенности:** Избегать кокосового масла, голодания

### LCHAD / TFP

- **Диета:** Строгое ограничение LCT (3-5% — только НЖК)
- **MCT:** 20-25% от калорийности
- **DHA:** Обязательна (профилактика ретинопатии)
- **Карнитин:** ⚠️ ПРОТИВОПОКАЗАН
- **Особенности:** Ежегодный осмотр офтальмолога

### VLCAD

- **Бессимптомная форма:**
  - MCT 10-15%, LCT 15-25%
  - Умеренное ограничение

- **Симптоматическая форма:**
  - MCT 20-25%, LCT 5-10%
  - Строгое ограничение
  - Карнитин в/в противопоказан

### GA2

- **Диета:** Низкожировая (20-25%), низкобелковая (8-10%)
- **MCT:** ⚠️ ПРОТИВОПОКАЗАНЫ
- **Рибофлавин:** 100-400 мг/сут (обязательно)
- **Особенности:** Оценить ответ на рибофлавин через 2-4 недели

## Часто задаваемые вопросы

### Почему калькулятор не открывается?

Калькулятор загружает данные из JSON-файлов, что требует HTTP-сервера. Откройте файл через локальный сервер (см. раздел «Запуск»).

### Как часто нужно пересчитывать диету?

- При изменении массы тела
- При смене возрастной группы
- При изменении состояния (заболевание, криз)
- Не реже 1 раза в 3-6 месяцев

### Можно ли использовать калькулятор для взрослых?

Да, калькулятор поддерживает расчёты для всех возрастов. Для взрослых применяются соответствующие нормы энергии.

### Что делать при метаболическом кризе?

1. Немедленно дать углеводы (раствор глюкозы/мальтодекстрина)
2. Исключить LCT на 24-48 часов
3. Кормить каждые 2 часа
4. При рвоте или невозможности питания — госпитализация
5. Использовать экстренный протокол из калькулятора

### Почему MCT противопоказаны при MCAD и GA2?

При MCAD нарушено окисление среднецепочечных жирных кислот (С6-С10), поэтому MCT не могут быть утилизированы и накапливаются.

При GA2 нарушено окисление всех жирных кислот, включая среднецепочечные.

### Зачем нужна DHA при LCHAD/TFP?

Пациенты с LCHAD/TFP не могут синтезировать DHA из предшественников из-за блока в окислении длинноцепочечных ЖК. Дефицит DHA приводит к:
- Пигментной дегенерации сетчатки
- Периферической нейропатии

Рекомендуются препараты DHA из микроводорослей (не содержат LCT).
