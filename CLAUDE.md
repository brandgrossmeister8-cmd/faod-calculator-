# Калькулятор диет FAOD

## Описание проекта
Калькулятор для расчёта диетотерапии при нарушениях окисления жирных кислот (FAOD).

## Структура проекта
- `js/calculator.js` — основная логика расчётов
- `js/app.js` — UI и отображение результатов
- `data/feeding_intervals.json` — данные по интервалам питания

## Расчёт интервалов питания

### Основная функция
`_calculateFeedingIntervals(ageInMonths, condition, diagnosis, clinicalForm)` в `js/calculator.js:523-650`

### Логика расчёта
1. Определение базовых интервалов по возрастной группе
2. Применение диагноз-специфичных коррекций
3. Применение коррекции по состоянию (интеркуррентное/криз)
4. Пересчёт количества кормлений

### Формула расчёта кормлений
```javascript
const dayHours = 14;      // 8:00 - 22:00
const nightHours = 10;    // 22:00 - 8:00
feedingsPerDay = Math.ceil(dayHours / dayInterval) + Math.ceil(nightHours / nightInterval);
```

### Типы коррекций по диагнозу

| Тип | Диагнозы | Описание |
|-----|----------|----------|
| `progressive` | MCAD | После 12 мес +2 ч к ночному интервалу (max 12 ч) |
| `stricter` | LCHAD, TFP, CACT | Сокращение интервалов, ограничение ночного |
| `conditional` | VLCAD, CPT2 | Зависит от клинической формы |
